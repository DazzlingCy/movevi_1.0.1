# MOVEVI
An immersive treadmill application.

This project is configured to be automatically deployed to GitHub pages via GitHub Actions.

## Development
```bash
npm install
npm run dev
```

## Deployment

The `.github/workflows/deploy.yml` takes care of deploying to GitHub pages upon pushing to the `main` branch. 
Make sure that "GitHub Actions" is selected as the source under **Settings > Pages > Build and deployment**.
